Что такое HTML?

HTML (HyperText Markup Language) — это язык разметки, используемый для создания структуры веб-страниц. HTML описывает структуру содержимого с помощью различных элементов, которые браузер отображает как заголовки, параграфы, списки, изображения и другие компоненты.

Пример HTML-кода:

Простая структура HTML-документа:

<!DOCTYPE html>
<html>
<head>
    <title>Заголовок страницы</title>
</head>
<body>
    <h1>Мой первый заголовок</h1>
    <p>Мой первый параграф.</p>
</body>
</html>

Основные элементы HTML

  • 1 Заголовки - определяются тегами от <h1> до <h6>
  • 2 Параграфы - создаются с помощью тега <p>
  • 3 Ссылки - создаются с помощью тега <a>
  • 4 Изображения - вставляются с помощью тега <img>
  • 5 Списки - бывают нумерованные (<ol>) и ненумерованные (<ul>)
  • 6 Таблицы - создаются с помощью тегов <table>, <tr>, <td>

HTML-страница состоит из набора элементов, которые образуют дерево документа (DOM). Каждый элемент может иметь атрибуты, которые предоставляют дополнительную информацию об элементе.

Советы по HTML

  1. Всегда используйте правильную структуру документа
  2. Проверяйте валидность своего HTML-кода
  3. Используйте семантические теги для улучшения доступности
  4. Тестируйте на разных устройствах и браузерах
  5. Следуйте современным стандартам HTML5

Семантические элементы HTML5

Современный HTML включает семантические элементы, которые лучше описывают содержимое:

  • <header> - верхний колонтитул
  • <nav> - навигация
  • <main> - основное содержимое
  • <section> - раздел документа
  • <article> - независимая часть контента
  • <footer> - нижний колонтитул
